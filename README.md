### Wasm Spider - оса паук

простой код с использованием структуры Spider под обшими характеристиками 

# Что делает программа

выводит данные созданного обьекта wasm_spider с его дальнейшими характеристиками и выводит их в консоль в красивом стиле

## Характеристика структуры паука

- Возраст 
- имя
- пол
- ширина и длина паука с учетом ног.

# Пример ввывода

```Wasm-Spider
        Name this spider is Ed
        Sex-Pol Male
        Spider x and y 4.5-5
        Spider age 1
```
